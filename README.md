# Profit vs Expenditure Analysis

## Overview
This project analyzes the relationship between expenditure and profit using Microsoft Excel. The goal is to explore how spending influences financial outcomes and identify patterns in efficiency.

## Dataset
The dataset consists of two variables:
- **Expenditure (€)** – amount of resources spent on advertisement
- **Profit (€)** – financial return generated

## Analysis Performed
- Calculated key metrics such as net profit, efficiency ratio, averages, minimum, and maximum values
- Created a **scatter plot** to visualize the relationship between expenditure and profit
- Added a **linear trendline** to identify the overall pattern
- Derived the trendline equation:
  
  **y = 11.76x + 20.68**

- Evaluated the strength of the relationship using **R² = 0.66**

## Key findings
- There is a **positive relationship** between expenditure and profit  
- On average, increasing expenditure leads to higher profit  
- The trendline suggests that each additional €1 of expenditure is associated with an increase of approximately €11.76 in profit  
- The R² value indicates a **moderate to strong correlation**, meaning expenditure explains a significant part of profit variation  
- Variations in data suggest differences in efficiency across observations  

## Tools Used
- Microsoft Excel  
  - Formulas (AVERAGE, MAX, MIN)  
  - Data visualization (scatter plot)  
  - Trendline and regression analysis
